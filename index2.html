About

My name is Kim Brandt and I'm a full stack web developer from Seattle.

I created my first website at the age of 12, and have always been strongly passionate about digital arts, the web, and photography. For the past 3 years I've been working as a graphic designer, and I'm now switching my focus to web development. I recently graduated from the Web Dev Immersive program at General Assembly, which was an intensive 3 month program which allowed me to sharpen up my skills and fully prepare me to be a full stack web developer. At GA I created several fully dynamic, responsive web apps both solo and on teams, which you can check out below.

PixaPost

An image sharing app geared towards artists or designers, where users can browse, save, or share inspiring images. Image feed is generated by pulling images via Tumblr API. Users can add images to their personal favorites gallery.

Technologies:  Express, Node.js, AJAX, Javascript, Sequelize, Facebook API/Oauth, Heroku, Bootstrap, HTML5, CSS3, Tumblr API

MyPetPal

Worked in a team of 4 to create a virtual pet app/game. User can perform various activities (feed, play, sleep, etc) and ‘pet’ responds accordingly. Created custom spritesheets and CSS animations.

Technologies: Angular, Javascript, Node.js, Express, MongoDB, Mongoose, HTML5, CSS3, Heroku

my.Life

A goal setting app created with Ruby on Rails. Includes login/signup authorization and CRUD features using a PostgreSQL database. Users can add and edit goals on their personal account, and can also leave notes/comments on each goal.

Technologies: Ruby on Rails, PostgreSQL, Materialize, HTML/CSS

Kitty Combat

An arcade style platformer game created with the Phaser.io framework. Included custom graphics and spritesheets.

Technologies: Phaser.io, Javascript, HTML, CSS
